## 🚀 About The Project

> **Disclaimer:** This project is a frontend recreation of the Mintlify landing page for educational purposes. It is not affiliated with or endorsed by Mintlify.

This project is a **desktop-first recreation of the Mintlify documentation-style landing page**, built entirely from scratch as part of a web development assignment. 

The primary focus was achieving **layout accuracy, structural fidelity, and premium readability**, closely matching the modern and sleek aesthetic of the original Mintlify front page. 

By building this without modern component libraries, it serves as a testament to deep, fundamental understanding of CSS layouts, flexbox, grid, and semantic HTML structure.

### Built With

Crafted with fundamental web technologies, adhering to strict constraints to demonstrate core frontend proficiency without relying on modern frameworks.

* ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
* ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

> **Constraints Followed:** Pure HTML/CSS only. No JavaScript, TailwindCSS, or pre-built UI components. Desktop-only layout optimized for widescreen viewing.

### Design & Typography

- **Primary Font:** Inter
- **Monospace Font:** Geist Mono
- Visual styling, spacing, and color palettes were meticulously mapped from the original Mintlify website.
- **Brand Assets:** Sourced via [Brandfetch](https://brandfetch.com/mintlify.com)

## ✨ Features Recreated

## 🛠 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You only need a modern web browser. For development, a local server is recommended to avoid CORS issues if you add external assets later.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Insohamdas/saas-documentation-landing.git
   ```
2. Navigate to the project directory
   ```sh
   cd saas-documentation-landing
   ```
3. Open `index.html` in your browser, or use a tool like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VSCode for hot reloading.
   ```sh
   # macOS
   open index.html
   ```
